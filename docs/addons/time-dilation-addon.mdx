# Time Dilation Addon

<Warning>This Addon is Experimental.</Warning>

The Time Dilation Addon is a Widgetbook utility designed for modifying
the scheduler's [`timeDilation`](https://api.flutter.dev/flutter/scheduler/timeDilation.html) property.

## Setup

Here's how to configure it:

```dart
TimeDilationAddon()
```
