# Alignment Addon

The Alignment Addon is a Widgetbook utility designed for modifying the alignment of the
use-case being previewed within the Workbench.

## Setup

Here's how to configure it:

```dart
AlignmentAddon(
  initialAlignment: Alignment.center,
),
```
